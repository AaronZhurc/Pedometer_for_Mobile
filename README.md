### 计步器 - Pedometer
*本应用程序原本为Android开发课程设计*  
*This application was originally designed for an Android development course*.
#### 测试环境 - Test Environment
Nokia 7 Plus, Android 10
#### 系统功能实现展示 - Demonstration of System Function
##### 登录界面 - Login Interface
<img width="253" height="474" alt="登录界面" src="https://github.com/user-attachments/assets/451b0086-c5db-4114-a248-c5d9a32158d6" />

点击蓝色框体即可免登录使用，输入用户名或密码，如没有输入系统会提示，如密码错误系统也会提示，点击登录即可进入。  
Click on the blue box can be used without logging in, enter the user name or password, if not entered the system will prompt, such as password error system will also prompt, click on the login to enter.
##### 点选生日后的注册界面 - Registration Screen after Clicking Birthday
<img width="266" height="477" alt="登录界面" src="https://github.com/user-attachments/assets/91552730-611f-4981-994b-8fc6d0d14cc4" />

此界面要求输入用户名、密码、身高、体重、性别、生日。也会对没有输入或密码格式错误或第二遍输入密码错误进行提示。点选性别会出现AlertDialog进行性别选择，生日则如图。  
This screen asks for a username, password, height, weight, gender, and birthday. It also prompts for no entry or wrong password format or wrong password entry the second time.
